## PARCOURS DE DEMONSTRATION

### Scénario 1 : Le Gestionnaire (Admin)

**Connexion :** demo@syndicpro.tn / Demo2026!Syndic

#### Etape 1 — Tableau de bord
- Montrer les KPI : trésorerie, taux de recouvrement, impayés, tickets
- Montrer le graphique encaissements vs dépenses (6 derniers mois)
- Point clé : "En un coup d'oeil, vous voyez la santé financière de votre résidence"

#### Etape 2 — Encaissements
- Aller dans **Encaissements**
- Montrer la liste des paiements par mois
- Enregistrer un paiement pour Fatma (A-02) — mois d'avril
- Point clé : "2 clics pour enregistrer un paiement"

#### Etape 3 — Dépenses
- Aller dans **Dépenses**
- Montrer les dépenses existantes avec catégories
- Ajouter une nouvelle dépense : "Produits d'entretien — 45 DT"
- Montrer la possibilité de joindre une facture (photo/PDF)
- Point clé : "Chaque dinar dépensé est tracé et justifié"

#### Etape 4 — Gestion des résidents
- Aller dans **Appartements** — montrer la liste des blocs et appartements
- Aller dans **Utilisateurs** — montrer les comptes résidents
- Montrer le code d'invitation : "Le résident s'inscrit lui-même avec ce code"
- Point clé : "Zéro paperasse, tout est numérique"

#### Etape 5 — Alertes impayés
- Aller dans **Alertes impayés**
- Montrer les résidents en retard (Fatma = 1 mois, Karim = 2 mois)
- Cliquer sur "Envoyer relance WhatsApp" pour Karim
- Point clé : "Fini les appels gênants, SyndicPro relance automatiquement"

#### Etape 6 — Tickets de support
- Aller dans **Tickets**
- Montrer le ticket de Karim (ascenseur bloqué — en cours)
- Répondre au ticket : "Le technicien passera demain à 10h"
- Point clé : "Les résidents signalent, vous suivez, tout est tracé"

#### Etape 7 — Annonces
- Aller dans **Annonces**
- Montrer l'annonce épinglée (coupure d'eau)
- Créer une nouvelle annonce : "Nettoyage des parties communes vendredi 18 avril"
- Point clé : "Communiquez avec tous les résidents en 1 clic"

#### Etape 8 — Trésorerie et rapports
- Aller dans **Trésorerie** — montrer le bilan mensuel
- Aller dans **Etats financiers** — montrer le rapport comptable complet
- Cliquer sur **Exporter en PDF** — télécharger le rapport
- Cliquer sur **Exporter en Excel** — télécharger les données
- Point clé : "Vos rapports sont prêts pour l'assemblée générale"

#### Etape 9 — Assemblée générale
- Aller dans **Assemblées** — montrer l'AG programmée
- Montrer l'ordre du jour avec les points à voter
- Montrer le vote en ligne (les résidents votent depuis leur téléphone)
- Montrer la génération automatique du PV en PDF
- Point clé : "Organisez vos AG et générez le PV en 1 clic"

#### Etape 10 — Assistant IA
- Aller dans **Assistant IA**
- Poser la question : "Quel est le taux de recouvrement ce mois ?"
- Poser : "Qui sont les résidents en retard de paiement ?"
- Poser : "Quel est le bilan financier du trimestre ?"
- Point clé : "Un assistant intelligent qui connaît votre résidence"

#### Etape 11 — Virements bancaires
- Aller dans **Virements en attente**
- Montrer une demande de virement soumise par un résident
- Confirmer le virement : saisir le montant reçu + frais bancaires
- Point clé : "Le résident déclare son virement, vous confirmez en 1 clic"

#### Etape 12 — Paramètres
- Aller dans **Paramètres**
- Montrer la configuration Konnect (paiement en ligne)
- Montrer la configuration WhatsApp (notifications automatiques)
- Montrer le code d'invitation résidents
- Montrer la gestion des caméras
- Point clé : "Personnalisez SyndicPro selon vos besoins"

#### Etape 13 — Appels de fonds
- Aller dans **Appels de fonds**
- Montrer comment créer un appel pour des travaux exceptionnels
- Chaque appartement a un quota calculé automatiquement
- Suivi des paiements par appartement
- Point clé : "Gérez les travaux exceptionnels avec transparence totale"

#### Etape 14 — Litiges
- Aller dans **Contentieux**
- Montrer comment ouvrir un dossier de litige contre un résident impayé
- Générer un accusé de réception / mise en demeure
- Point clé : "Passez du rappel amiable au juridique en quelques clics"

#### Etape 15 — Intervenants
- Aller dans **Intervenants**
- Montrer l'annuaire des prestataires (plombier, électricien, nettoyage)
- Ajouter un nouveau prestataire avec ses coordonnées
- Point clé : "Votre carnet d'adresses de prestataires, toujours accessible"

#### Etape 16 — Badges et accès
- Aller dans **Badges**
- Montrer les badges attribués aux résidents
- Montrer le journal d'accès (entrées/sorties)
- Point clé : "Contrôlez les accès de votre résidence"

#### Etape 17 — Messagerie directe
- Aller dans **Messagerie**
- Montrer les conversations avec les résidents
- Envoyer un message à Fatma : "Bonjour, pensez à régulariser votre situation"
- Point clé : "Communiquez en privé avec chaque résident"

### Scénario 2 : Le Résident

**Connexion :** mohamed@resident.tn / Resident2026!

#### Etape 1 — Espace résident
- Montrer les KPI : redevance mensuelle, mois impayés, crédit
- Montrer l'historique des paiements (vue factures)
- Point clé : "Le résident voit tout ce qu'il doit et ce qu'il a payé"

#### Etape 2 — Payer en ligne
- Cliquer sur "Payer en ligne" (Konnect ou Flouci)
- Montrer le choix du mois et le montant
- Montrer le paiement groupé (plusieurs mois d'un coup)
- Point clé : "Payez en 30 secondes depuis votre téléphone"

#### Etape 3 — Déclarer un virement
- Cliquer sur "Déclarer un virement bancaire"
- Remplir : mois, montant, référence, photo du reçu
- Soumettre — le gestionnaire reçoit une notification WhatsApp
- Point clé : "Même par virement, tout est tracé numériquement"

#### Etape 4 — Signaler un problème
- Aller dans **Tickets** (via la sidebar)
- Créer un ticket : "Interphone de l'entrée principale ne fonctionne plus"
- Montrer le suivi (statut, réponse de l'admin)
- Point clé : "Signalez un problème, l'admin est notifié instantanément"

#### Etape 5 — Consulter les annonces
- Voir les annonces sur l'espace résident
- Montrer l'annonce épinglée (coupure d'eau)
- Point clé : "Restez informé de la vie de votre résidence"

#### Etape 6 — Voter en assemblée
- Si une AG est en cours de vote
- Le résident vote "Pour", "Contre" ou "Abstention" sur chaque point
- Point clé : "Votez depuis chez vous, pas besoin de se déplacer"

#### Etape 7 — Messagerie
- Envoyer un message privé à l'admin
- Point clé : "Communiquez directement avec votre gestionnaire"

#### Etape 8 — Profil
- Modifier son nom, téléphone
- Changer son mot de passe
- Point clé : "Le résident gère son propre compte"

### Scénario 3 : Auto-inscription d'un résident

#### Etape 1
- Aller sur syndicpro.tn/register-resident
- Le résident entre le **code d'invitation** de sa résidence (fourni par l'admin)
- Il choisit son appartement dans la liste
- Il crée son mot de passe

#### Etape 2
- Il se connecte et accède immédiatement à son espace
- Il voit ses factures, peut payer, signaler des problèmes

- Point clé : "L'admin n'a même pas besoin de créer le compte — le résident s'inscrit seul"

## POINTS CLES A METTRE EN AVANT PENDANT LA DEMO

1. **Simplicité** : Aucune installation, tout se fait depuis le navigateur (mobile ou PC)
2. **Gain de temps** : Plus de cahiers, plus d'Excel, tout est automatisé
3. **Transparence** : Chaque résident voit exactement ce qu'il doit et ce qu'il a payé
4. **Paiement en ligne** : Konnect + Flouci — le résident paie en 30 secondes
5. **Notifications** : WhatsApp + Push + Email — tout le monde est informé
6. **Rapports** : PDF et Excel en 1 clic pour les AG et le suivi
7. **Assistant IA** : Posez vos questions, obtenez des réponses instantanées
8. **Sécurité** : Chaque résidence est isolée, données protégées
9. **Essai gratuit** : 3 mois sans engagement, sans carte bancaire
10. **Support** : Contact direct via contact@syndicpro.tn
